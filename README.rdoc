== BioLoader

Collection of tools for loading, synchronizing and updating raw Biological data from public ftp repositories into the data warehouse.  BioLoader will support delta downloads or incremental updates, with API access. In addition, bioftpalerts provides social alerts (Twitter, Facebook and FriendFeed) when Bio FTP servers update their data. 

Learn more about delta downloads or incremental updates,
http://www.abhishek-tiwari.com/2010/10/delta-downloads-for-big-data-in-bioinformatics.html

== Dependencies

* Python 2.5+
* tweepy
* ftputil
* rsync
